# Dark Mode for Kingwood.com

A Firefox extension that gives Kingwood.com a clean, easy-on-the-eyes dark mode. Covers all pages: forums, threads, classifieds, news, and more.

![Firefox](https://img.shields.io/badge/Firefox-109%2B-orange)

---

## How to Install (Step by Step)

### Step 1: Download the Extension

1. Go to this page: **https://github.com/zman-six/kingwood-dark-mode**
2. Click the green **"Code"** button near the top of the page
3. Click **"Download ZIP"** from the menu that appears
4. Save the file to your computer (it will probably go to your **Downloads** folder)

### Step 2: Unzip the File

1. Go to your **Downloads** folder
2. Find the file called **kingwood-dark-mode-main.zip**
3. **Right-click** on it and choose **"Extract All"** (Windows) or double-click it (Mac)
4. This will create a folder called **kingwood-dark-mode-main** — remember where it is

### Step 3: Open Firefox's Add-on Page

1. Open **Firefox**
2. In the address bar at the top, type: **about:debugging** and press **Enter**
3. On the left side, click **"This Firefox"**

### Step 4: Load the Extension

1. Click the **"Load Temporary Add-on"** button
2. A file picker window will open
3. Navigate to the **kingwood-dark-mode-main** folder you unzipped
4. Click on the file called **manifest.json** (not a folder, the file itself)
5. Click **Open**

### Step 5: Visit Kingwood.com

1. Go to **www.kingwood.com** — it should now be in dark mode
2. To **turn dark mode off**, click the moon icon in your Firefox toolbar
3. Click it again to **turn it back on**

---

## Important Note

This is loaded as a **temporary** add-on. That means **it will disappear when you close Firefox**. You'll need to repeat **Steps 3 and 4** each time you restart the browser.

Once the extension is approved on the Firefox Add-ons store, you'll be able to install it permanently with one click. Stay tuned!

- Easy-to-read light text

- Toggle on/off with one click
